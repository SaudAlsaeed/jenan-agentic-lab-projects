# Jenan Lab Hub

Multica project folder for **Jenan Lab Hub** — team command center for the Jenan-agentic-lab workspace.

| Asset | Path | Stack |
| --- | --- | --- |
| Frontend | `web/` | React 19 + Vite + Tailwind |
| Backend | `api/` | Node.js + Express |

From repo root:

```bash
pnpm dev:jenan-lab-hub:web
pnpm dev:jenan-lab-hub:api
```

See `web/README.md` and `api/README.md` for package-level details.
