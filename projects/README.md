# Projects

Each subdirectory is one Multica project. Folder name should match the project slug (kebab-case).

| Folder | Multica project |
| --- | --- |
| `jenan-lab-hub/` | Jenan Lab Hub |

To scaffold a new project:

```
projects/my-new-project/
  README.md
  web/
  api/
```

Then create the Multica project pointing at this repo and set its description to: `Monorepo path: projects/my-new-project/`
